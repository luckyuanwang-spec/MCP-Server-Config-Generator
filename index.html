❓ Frequently Asked Questions

What is MCP (Model Context Protocol)?▼
MCP is an open protocol by Anthropic that allows AI assistants (like Claude, ChatGPT) to connect to external tools and data sources. It standardizes how AI models interact with APIs, databases, file systems and more. Think of it as "USB for AI" - a universal plug standard.
Where do I put the generated config?▼
Each client has its own config location:
Claude Desktop: Settings → Developer → Edit Config (claude_desktop_config.json)
Cursor: Settings → MCP → Edit mcp.json
VS Code: ~/.vscode/mcp.json (or use the Copilot extension settings)
Windsurf: ~/.windsurf/mcp.json
Do I need to install anything?▼
Most MCP servers run via npx (Node.js) or uvx (Python). You need Node.js 18+ or Python 3.10+ installed. The npx -y flag auto-downloads the server package on first run.
How is this tool different from mcp.so or smithery?▼
mcp.so and smithery are MCP server directories - they help you discover servers. This tool is a config generator - it helps you create the correct JSON configuration file to connect those servers to your AI client. Use both together: discover on mcp.so, configure here.
Why does Windows need cmd /c wrapper?▼
On Windows, npx is actually npx.cmd, which Claude Desktop may not find directly. Wrapping with "command": "cmd", "args": ["/c", "npx", ...] ensures the command runs correctly on all platforms.
Is my data sent to any server?▼
No. All generation happens in your browser. Nothing is sent to any server. Your config data stays on your machine.
