# MCP Server Config Generator

Free online tool to generate MCP server configuration files for Claude Desktop, Cursor, VS Code and Windsurf.

## Features

- **17 server templates**: Filesystem, Memory, Fetch, Git, SQLite, GitHub, Brave Search, Puppeteer, Slack, Google Maps, PostgreSQL, Playwright and more
- **3 modes**: Template Builder, Custom Config, Multi-Server Bundle
- **4 clients**: Claude Desktop, Cursor, VS Code (Copilot), Windsurf
- **Windows support**: Auto cmd /c wrapper
- **One-click copy & download**
- **Bilingual**: English + Chinese
- **100% client-side**: No data sent to servers

## Deploy

1. Push to GitHub
2. Import to Vercel
3. Submit to Google Search Console
